What are patches?

Patches are bits of JavaScript code that can be executed during runtime, and do not affect any of Textual's internal files. They usually modify, add, or remove pieces of data saved to localStorage.

How do I request a patch?

Patches can be obtained by making queries to /api?m=get&rt=patch&fn=patches.json.

Request Parameters

Parameter Name Verbose Name Accepted Values
m method
  • get
  • post
rt resource type
  • file
  • patch
